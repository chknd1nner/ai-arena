# AI Arena Frontend

React-based visualization frontend for AI Arena.

## Installation

```bash
npm install
```

## Development

```bash
npm start
```

Runs on `http://localhost:3000` with proxy to backend on `http://localhost:8000`.

## Build

```bash
npm run build
```

Creates optimized production build in `build/` directory.

## Future Features

- Real-time match visualization with Canvas 2D
- Replay viewer with playback controls
- Thinking tokens display side-by-side
- Match statistics and analytics
